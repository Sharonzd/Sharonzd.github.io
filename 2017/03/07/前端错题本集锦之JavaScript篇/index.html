前端错题本集锦之JS篇

发表于 2017-03-07   |   分类于 前端笔试   |     |   阅读次数

从中学时就养成了用错题本收集错题的习惯, 即便工作了也一直沿用至今, 所以打算写一个系列文章, 将平时前端开发中遇到的坑或者小知识点分享出来, 希望能对大家也有所帮助。知识点杂多, 暂且粗略的分类为HTML篇、JavaScript篇、CSS篇、Vue篇, 待以后不断扩充之后再做更细致的分类。此篇为其中的JS篇。

  1. 绑定事件的时候，元素还没有渲染出来，无法获取到，就会导致元素无法正确绑定事件。
    解决办法：使用点击事件的代理
    代码：$(‘body’).on(‘click’,’element’,function(){execute})
    ready只是页面dom的加载，如果有些元素是通过模板异步获取的，ready就不管用了。直接给元素绑定的时候，元素还没有生成，事件就绑定不上，如果用body来代理，给body绑定点击事件，点击body的时候，检查点击的具体目标是不是element，是就执行。

  2. js控制链接跳转时，避免跳转错误，可以先清空href，再写href。location.href=””; location.href=”xxx”。（不记得原因了）

  3. jquery可以给多个元素直接绑定事件 $(“.class”).on(“click”,function(){})。原生必须单个绑定事件。buttons[i].onclick = function(){}

  4. HTMLElement.innerHTML += “xxxxx”;会破坏它本来的节点结构。会导致里面的子节点变成简单的字符串，从而无法绑定事件。

  5. 涉及到用JS对图片的处理。获取图片尺寸，获取图片内容等。需要判断图片是否加载完，有以下几种方法
    a. onload方法：img.onload=function(){ console.log(“loaded”) }
    b. 轮询complete属性：

    function imgLoader(img, callback) {
    var timer = setInterval(function () {
        if (img.complete) {
            callback(img);
            clearInterval(timer);
        }
    }, 50)
}
imgLoader(imageElem, function () {
    console.log("completed");
    ctx.drawImage(imageElem,10,10);
    ctx.drawImage(imageElem,120,10,100,120);
    ctx.drawImage(imageElem,20,20,100,50,250,10,100,120);
});

    c. onreadystatechange(仅仅支持ie）

  6. 涉及到页面渲染切换的问题，js执行可能过快，页面来不及渲染，此时需要使用setTimeout()
    a. this的指向会变。如果要用到this，应该在外面令that = this
    b. 如果只是为了解决渲染问题。setTimeout的值为0也可以，0也能使其延迟。

  7. 点击元素增加/减少css样式
    a. $(element).css(‘style-name’,’style-value’)，由此添加的css样式会成为内联样式。
    b. $(element).css(‘style-name’,’’)使用这个来移除对应的样式。
    c. $(element).removeAttr(‘style’)或者直接移除所有的内联样式
    d. 将样式写在css的class里，然后点击addClass/removeClass

  8. 设置元素位置偏移
    a. $().offset().top 获取距离顶部距离 $().offset().left 获取距离左侧距离
    b. $().offset({top:’’,left:’’}) 设置元素偏移
    c. $().offset(function(selectorIndex,oldOffset){return {top:’’, left:’’}) 利用函数所已知参数（类似于function(e)）来设置元素偏移

  9. 设置元素滚动高度
    a. $().scrollTop() 获取滚动高度
    b. $().scrollTop(offset) 设置滚动高度。
    c. 注意，只有滚动的元素才可以获取并设置。匹配的是元素的滚动条相对于其顶部的偏移。如果是整个页面超出窗口导致的滚动，那么只有$(window)才能获取并设置。

  10. 获取元素高度 $().height()

  11. 安卓渲染问题/动画渲染问题。某些样式显示不出来，或者动画过于卡顿。可以试试触发渲染加速
    scale3d 或者 translateZ(0)
    http://www.jianshu.com/p/133085f0eb76
    http://www.jianshu.com/p/50472944c3e0
    http://www.w3cplus.com/css3/introduction-to-hardware-acceleration-css-animations.html

  12. 对象只能用属性取元素，不能数组索引。如console.log(photo[“photoexterior”]);//不能用photo[0]来取对象中的第一个元素。

  13. es6用法：不定参数…可以接收一串由,分隔的参数，然后将它们转换为一个数组，赋值给…后的变量。
    a. 用法1.
    return{

    poiName,
logoImage,
...getHeaderImage(photo),
...processPoiNameBlock(avgScore),

    };

  14. / global lib/可以用来声明全局变量，避免js检测工具检测时判断全局变量为错误语法

  15. 严格比较运算符（===，!==）。转换类型比较运算符(==,>,<,>=,<=)
    a. 数字和字符串，字符串转换成数字
    b. 布尔类型和*，布尔类型会被转换为1或0
    c. 对象和数字/字符串，对象通过valueOf/toString转换为数字或字符串，失败则报错
    d. 对象与对象比较时，仅当引用相同，结果才为true

    i. 字符串对象是对象，不是字符串
ii. new String('foo') == new String('foo')  //false
iii. new String('foo') == 'foo'  //true

  16. jq可以链式操作，以简化代码。

  17. hasAttribute是HTML DOM的方法，判断是否有某属性; attr()是JQuery的方法，设置或返回被选元素的属性值。所以判断是否有属性有以下两种方法：

    1. if(elem.hasAttribute(“xxx”)){ // 有属性}else{//没有属性}
    2. if(attr(“xxx”))。（有可能是有属性，但是属性值为false，所以应该是判断attr(“”)且不为false或者typeof attr(“”) 不为undefined。（然而，貌似现在好像不存在这种情况了。。））

  18. 取得data-xxx的方法有两种：
    $.data(xxx)
    $.attr(“data-xxx”)

  19. jq元素用[]取后，会变为dom元素，用eq()取仍为jq元素

  20. click/addEventListener的区别：
    click点击只能一次，addEventListener可以绑定多次

  21. if（值===变量） 值放在===左边，以避免变量为undefined

  22. 屏幕翻转时，避免批量渲染，可以通过devicemotion事件模拟onbeforeorientationchange

  23. 修改大于小于的时候注意反向。如a > b 的反向是 a <= b

  24. $()即便元素不存在，也会返回空数组。空数组和空对象都为true。所以不能用$()的方式判断是否有元素。应该是
    if($(‘xx’) && $(‘xx’).length > 0){}

  25. display为none的元素，高度为0，获取display为none的元素高度方式有以下两种：

    1. visibility:hidden + position:absolute
    2. Jquery的$().height()能正确取到display为none的元素的高度

  1. 通过页面或者url参数传过来的数字如果需要用,需要手动使用+或者parseInt()转换为数字,其中+更高效。

  2. es6的import是singleton模式，如果import两次，只会执行一次。

  3. npm install fecs -g 安装代码检查工具，本地检查代码风格

  4. jq的多个事件绑定写法：
    on({
    “click”:function(){},
    “scroll”:function(){}
    })

  5. 选出特定dom绑定点击事件的两种处理方式

    1. 通过stoppogation
    2. 判断event.target

  6. new Date() - ‘毫秒数时间戳’（字符串或者数字） // 毫秒数时间戳 // 默认会把new Date()转化为time
    但是 new Date() + ‘毫秒数时间戳’ // 字符串拼接

  • new Date() // 1488954511789 new Date().toString() // “Wed Mar 08 2017 14:28:18 GMT+0800 (CST)”

  1. 不涉及到具体时分秒的时间戳的比较，可以将时间转化为当天0点的时间戳，以保证后面计算和对比的准确性

  2. 轮播式的左右移动页面的方式有很多，如absolute且动态修改left值，如动态修改margin-left
