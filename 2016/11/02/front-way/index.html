30天前端从入门到上手

发表于 2016-11-02   |     |   阅读次数

周围常常有人问我, 想要学前端, 不知道如何下手。在次分享一下我学习前端的路线, 希望能够帮助到大家。欢迎推荐以及讨论。

7天入门

1. 入门工具

sublime/webstorm
http://www.imooc.com/learn/40 (建议2小时)

2. 入门视频：html+css基础课程 http://www.imooc.com/learn/9 （建议用时1天）

3. js入门视频：

js深入浅出 http://www.imooc.com/learn/277 （建议用时1.5天）

4. 入门必读：

a. javascript编程艺术（薄，建议用时1天）；
b. javascript启示录（薄，罗列js重点，建议1天）
a. html5揭秘（薄，建议用时1天）（ http://diveintohtml5.info/ 英文版在线）
b. 精通css高级web标准解决方案（薄，建议用时1.5天）

21天上手

a. javascript语言精粹（薄，经典，知识量大，可多看几遍，不理解的可跳过，建议2天）
a. html5权威指南（厚，全，建议7天）
b. css权威指南（厚，全，建议5天）
c. javascript权威指南/javascript高级程序设计（厚，全，建议7天）

注：

  1. 此时间建议是建立在全天都全身心学习的前提下。1day ≈ 14hour+
  2. 看书/看视频不理解的地方不要花太多时间纠结，标记或者记录一下。之后看书的时候肯定会遇到，然后再回头理解。

##书籍推荐

  1. 高性能javascript
  2. javascript面向对象编程指南
  3. javascript模式
  4. javascript dom高级程序设计
  5. 高性能网站建设指南
  6. ……（此处省略20本）
