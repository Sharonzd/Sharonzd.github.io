30天前端从入门到上手

发表于 2016-10-31   |   分类于 方法   |     |   阅读次数

    身边常常有后端同学或者学弟学妹问我, 想要自学前端, 不知道如何下手。网上分享的资料大多学习曲线过于陡峭,技术过于繁杂, 常常让初学者看懵甚至吓退了。在此分享一下我学习前端的入门路线,希望能够帮助到一些。欢迎推荐以及讨论或者寻求资源。

7天入门

入门工具:

根据自己的需求和喜好自行研究并选择。(建议2小时)

  1. sublime 下载页 学习视频
    优点: 小巧, 流畅, 免费, 定制型强
    缺点: 自己按需安插件比较麻烦
  2. webstorm 下载页
    优点: 功能强大, 方便
    缺点: 占内存, 收费（可百度序列码 / 学生可以免费注册使用）

入门视频：

  1. html+css基础课程（建议用时1天）
  2. js深入浅出（建议用时1.5天）

入门必读书籍：

  1. javascript编程艺术（薄，建议用时1天）；
  2. javascript启示录（薄，罗列js重点，建议1天）
  3. html5揭秘（薄，建议用时1天）（英文版在线）
  4. 精通css高级web标准解决方案（薄，建议用时1.5天）

21天上手

读书

  1. javascript语言精粹（薄，经典，知识量大，可多看几遍，不理解的可跳过，建议2天）
  2. html5权威指南（厚，全，建议7天）
  3. css权威指南（厚，全，建议5天）
  4. javascript权威指南/javascript高级程序设计（厚，全，建议7天）

实践

  1. 参照学习视频实现一些有意思的, 比如七夕

  2. 做1个静态前端网站, 比如电商网站, 可使用Bootstrap、JQuery等框架减少开发量。

    注：

    1. 此时间建议是建立在全天都全身心学习的前提下。1day ≈ 14hour+
    2. 看书/看视频不理解的地方不要花太多时间纠结，标记或者记录一下。之后看书/实践的时候肯定会遇到，然后再回头理解。

其他书籍推荐

  1. 高性能javascript
  2. javascript面向对象编程指南
  3. javascript模式
  4. javascript dom高级程序设计
  5. 高性能网站建设指南
  6. 锋利的JQuery
  7. ES6标准入门
  8. CSS Secrets
  9. CSS禅意花园
  10. ……
